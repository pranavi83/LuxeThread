# Aero. Zero-Gravity Design System

An experimental interactive landing site that simulates a zero-gravity environment. The design is extremely clean, minimalist, and uncluttered, utilizing a shared zero-g liquid-glass design system and custom JS video crossfading.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v4, Custom CSS Variables
- **Animations**: Framer Motion (Continuous Levitation & Parallax)
- **Fonts**: Instrument Serif & Inter

## Features

- **Zero-G Liquid-Glass**: Subtle frosted glass components simulating floating UI elements.
- **Antigravity Mechanics**: Randomized, continuous levitation using Framer Motion.
- **Abstract Hero**: Custom FadingVideo wrapper implementing seamless video crossfades using `requestAnimationFrame`.
- **The Void**: Asymmetrical sprawling CSS grid utilizing negative space.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
```
